# Tableau Business Analytics Dashboard Portfolio

A portfolio of Tableau dashboard projects created for Business Analytics with Dashboards coursework. These projects focus on dashboard design, chart selection, KPI storytelling, geographic analysis, interactive filters, and written business insights.

The goal of this repository is to show practical Tableau skills: building dashboards from structured datasets, choosing appropriate visualizations, formatting dashboards for business users, and explaining insights clearly.

---

## Portfolio Projects

### 1. Financial Dashboard

A business analytics dashboard focused on financial/consumer complaint performance indicators. The dashboard uses KPIs, maps, bar charts, donut charts, filters, and written insights to summarize complaint volume and response patterns.

**Skills demonstrated:**

- KPI cards for total complaints, responded cases, and in-progress cases
- Donut chart design for consumer dispute summaries
- State-level map visualization
- Bar charts with labels and business insights
- Filters and dashboard-level formatting
- Business insight writing for non-technical audiences

**files:**

```text
financial-dashboard/
├── workbook.twbx
├── report.pdf
└── screenshots/
```

### 3. California Governor Election Dashboard

An interactive Tableau dashboard prototype showing candidate-wise and county-wise election-pattern analysis. The dashboard includes date filters, candidate vote comparisons, party color encoding, and a county-level map.

**Important note:** This dashboard was created as an educational Tableau dashboard-building project. The dataset should be interpreted as class/project data for visualization practice, not as official certified election reporting (although it is rather accurate).

**Skills demonstrated:**

- Interactive date filters for June 4, June 9, and combined views
- Candidate-wise bar chart comparisons
- County-wise geographic map
- Party-based color encoding
- Dashboard layout and filtering
- Written analysis of candidate rankings and geographic patterns

**files:**

```text

├── Governor Election Last.twbx
├── Governor's Election California.pdf
└── screenshots/
```

## How to View the Dashboards

To view the Tableau workbooks:

1. Download the `.twbx` file from the relevant project folder.
2. Open it with Tableau Desktop or Tableau Public.
3. Review the included PDF report and screenshots for the main dashboard outputs and written insights.

If Tableau is not installed, the screenshots and PDF reports provide a quick preview of the final dashboards and analysis.
